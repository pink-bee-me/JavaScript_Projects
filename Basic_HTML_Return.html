Using The Return Statement in JavaScript Functions

Define Return Statement:
Rule #1 - Every function in JavaScript [returns] undefined
unless otherwise specified. A value can be specified in a function with the return statement.
Rule #2 - The [return] statement (when executed) ends function execution immediately...meaning any code after the return statement will not be executed.
Rule #3 - The [return] statement returns a value to the function caller.
Note: The [return] statement can be used to interrupt or end a function at a specific point,...for example, if a user is required to input specific data type but doesn't follow directions you can use return to end the function. The [return] statement can also be used to interrupt a loop at a specified point. Also, you can return a function from within a function...but, this has to do with closures and I haven't learned that yet.
